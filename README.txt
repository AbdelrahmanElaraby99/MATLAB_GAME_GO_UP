To have the fun playing with our game do the following:
1-open the 'Game.m' file using matlab
2-hit the 'run' button above in the matlab.
3-voila! you can play using the keyboard arrows, don't touch the falling obstacles to stay alive!
   you can press 'q' on the keyboard anytime to quit the game.
